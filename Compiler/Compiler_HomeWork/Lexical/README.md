# leaxical

## lexical analysis code of complier homework

## how to complier
```
git clone git@github.com:LostMoonkin/Complier_Homework.git
cd Complier_Homework/lexical && cmake .
make
```
`Require cmake version >= 3.6`

## how to run the code
```
./leaxical filename
```
example: `./leaxical test.txt`
