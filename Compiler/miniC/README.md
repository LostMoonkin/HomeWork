# miniC
## a minimum C complier

### How to complier the code
```
git clone git@github.com:LostMoonkin/miniC.git
cd miniC && cmake .
make
```

### How to use miniC

```
./miniC [-s] [-d] yourcode.c
like
./miniC test/test.c
[-s] just complier
[-d] debug the code
```
